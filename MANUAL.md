# Conductor キーボード 取扱説明書

## キーマップ概要

40キー スプリット構成（左20 + 右20）。レイヤーは **ホールド** で切り替え。

凡例：`▽` = 透過（下レイヤー参照） / `×` = 無効

---

## レイヤー一覧

| # | 名前 | 切り替え方法 |
|:---:|---|---|
| 0 | デフォルト | 常時 |
| 1 | FUNCTION | TAB長押し |
| 2 | NUM | BSPC長押し（右） |
| 3 | ARROW | ENT長押し |
| 4 | MOUSE | トラックボール操作で自動 |
| 5 | SCROLL | MB1+MB2 同時押し |
| 6 | Bluetooth | FUNCTION中のBTキー |

---

## Layer 0 — デフォルト

| | 左1 | 左2 | 左3 | 左4 | 左5 | | 右1 | 右2 | 右3 | 右4 | 右5 |
|---|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| 上段 | Q | W | E | R | T | | Y | U | I | O | P |
| 中段 | A | S | D | F | G | | H | J | K | `-` | L |
| 下段 | Z | X | C | V | B | | N | M | 左click | 右click | `?` / ⇧ |
| 親指 | ⇧ | GUI | ALT | BS / CTL | SPC | TAB / L1 | BS / L2 | ENT / L3 | VOL- | `/` |

- `BS / CTL` … タップ=Backspace、長押し=Ctrl
- `TAB / L1` … タップ=Tab、長押し=Layer1
- `BS / L2`  … タップ=Backspace、長押し=Layer2
- `ENT / L3` … タップ=Enter、長押し=Layer3
- `? / ⇧`   … タップ=`?`、長押し=右Shift

## Layer 4 — MOUSE（自動）

トラックボールを動かすと自動で入る。全キー透過（通常通り入力可能）。

---

## Layer 5 — SCROLL（MB1+MB2 同時押し）

MB1+MB2 を押している間、トラックボールがスクロールになる。

---

## Layer 6 — Bluetooth

| | 左1 | 左2 | 左3 | 左4 | 左5 | | 右1 | 右2 | 右3 | 右4 | 右5 |
|---|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| 上段 | ▽ | ▽ | ▽ | ▽ | ▽ | | BT0 | BT1 | BT2 | BT3 | ▽ |
| 中段 | ▽ | ▽ | ▽ | ▽ | ▽ | | 切断0 | 切断1 | 切断2 | 切断3 | ▽ |
| 下段 | ▽ | ▽ | ▽ | ▽ | ▽ | | ▽ | ▽ | ▽ | ▽ | ▽ |
| 親指 | ▽ | ▽ | ▽ | ▽ | ▽ | ▽ | ▽ | BOOT | CLR | CLRALL |

- `BT0〜3` … プロファイル切り替え
- `切断0〜3` … 指定プロファイルの接続切断
- `CLR` … 現在のプロファイルのペアリング削除
- `CLRALL` … 全ペアリング削除
- `BOOT` … ブートローダーモード（ファームウェア書き込み用）

## コンボ一覧

| キー同時押し | 動作 |
|---|---|
| Q + W | ESC |
| MB1 + MB2 | スクロールレイヤー（Layer 5）ON |
| MINUS + L | 中クリック（MB3） |

## LED表示

### バッテリー残量（起動時・BLE接続時に点滅）

| 残量 | 色 |
|---|---|
| 50%以上 | 緑（点滅） |
| 10〜50% | 黄（点滅） |
| 10%以下 | 赤（点滅） |
| 5%以下 | 赤（常時点灯） |

### 充電中

| 状態 | 色 |
|---|---|
| 充電中・50%以上 | 緑（常時点灯） |
| 充電中・10〜50% | 黄（常時点灯） |
| 充電中・10%以下 | 赤（常時点灯） |
| 充電完了・未接続 | 通常表示に戻る |

### レイヤーインジケーター

| レイヤー | 色 |
|---|---|
| 0（デフォルト） | 消灯 |
| 1（FUNCTION） | 青 |
| 2（NUM） | 緑 |
| 3（ARROW） | 黄 |
| 4〜6 | 消灯 |

## Bluetooth 接続

### 接続先の切り替え

`[TAB]`（Layer 1）を開いて `BT0〜BT3` で切り替え。
最大4台まで登録可能。

### 新しいデバイスとペアリング

1. 未使用のプロファイル（BT0〜3）を選択
2. PC/スマホ側でBluetoothデバイスを検索
3. `conductor` が表示されたら接続

### ペアリングのリセット

Layer 6 → `CLR`（現在のプロファイルのみ）
または `CLRALL`（全削除）

## ファームウェア更新手順

1. GitHubのActionsからビルド済みファームウェアをダウンロード
2. キーボードをUSBで接続
3. ブートローダーモードに入る（Layer 6 → `BOOT`）
4. `MONOKEY_L_SEEEDUINO_XIAO_BLE-zmk.uf2` を左側に書き込み
5. 同様に右側も書き込み
6. `settings_reset` は **ペアリングが消えるため原則不要**
